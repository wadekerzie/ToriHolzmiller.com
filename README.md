## ğŸ“ Customization

### Update Contact Information

Edit the following files to customize with actual information:

1. **ProfileHeader.jsx** - Update name, bio, and tagline
2. **LinkHub.jsx** - Update social media links, email, phone
3. **SocialFooter.jsx** - Update social media URLs
4. **CalendarModal.jsx** - Add your booking widget URL
5. **index.html** - Update meta tags and domain

### Replace Images

- `/public/headshot.jpg` - Replace with actual professional headshot
- `/public/salon-bg.jpg` - Replace with actual salon photo (optional)

### Email Integration

The new client form needs an email service to function. See `EMAIL_INTEGRATION.md` for detailed setup instructions using:
- EmailJS (recommended)
- Formspree
- Google Forms
- Custom backend
